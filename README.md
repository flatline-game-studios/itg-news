# itg-news

A simple repository to get the patch news v4. 


# Codigos de localizacion

 - `en` > English
 - `es` > Spanish
 - `fr` > French
 - `ja` > Japanese
 - `ko` > Korean
 - `zh` > Chinese (Simplified)
 - `zh-hant` > Chinese (Traditional)
 - `pt-br` > Portuguese (Brazil)
 - `ru-ru` > Russian
